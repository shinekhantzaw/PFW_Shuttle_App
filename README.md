## üöÄ Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- MongoDB installed locally
- Git

### Installation

1. Clone the repository
```bash
git clone https://github.com/shinekhantzaw/PFW_Shuttle_App.git
cd .\PFW_Shuttle_App\
```

2. Install Frontend Dependencies
```bash
cd client
npm install -y
```

3. Install Backend Dependencies
```bash
cd ../server
npm install -y
```

Terminal 1 - Backend:
```bash
cd server
npm run dev
```

Terminal 2 - Frontend:
```bash
cd client
npm run dev
```
